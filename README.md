# Finessa - Landing Page

A modern, responsive landing page for Finessa, a financial education platform designed specifically for women.

## Features

- **Modern Design**: Clean, professional design with a green/nature color scheme
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle animations and transitions for better user experience
- **Accessible**: Semantic HTML and proper ARIA labels
- **Fast Loading**: Optimized CSS and JavaScript
- **Interactive Elements**: Smooth scrolling, mobile menu, and form handling

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - JavaScript for interactivity

## Sections

1. **Hero Section** - Main introduction with call-to-action buttons
2. **About Section** - Overview of Finessa
3. **Program Section** - Details about the Hybrid Program
4. **Benefits Section** - Key benefits of the program
5. **Who is it for Section** - Target audience information
6. **Contact Section** - Contact form for inquiries
7. **Footer** - Additional links and information

## Color Scheme

- Primary Green: `#2d8659`
- Secondary Background: `#f4f7f6`
- Text Dark: `#1a1a1a`
- Text Light: `#666666`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Usage

Simply open `index.html` in a web browser to view the landing page. No build process or dependencies required.

## Customization

To customize the content:
- Edit the text in `index.html`
- Modify colors in the `:root` variables in `styles.css`
- Adjust animations and interactions in `script.js`
